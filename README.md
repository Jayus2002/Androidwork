# 接口
| 接口名称   | 作用 |  
| :----- | :--: |
| login |  登录  |  
| register |  注册装修公司  | 
| registerZX | 同上pc端使用(权限管理)   |
| registerJc |注册建材市场  |
| registerSjs| 注册设计师|
| judeg_login|判断是否登录|
| logout| 退出账号|
| person| 返回个人数据 |
|data|返回装修公司，建材市场或者设计师列表 需要form参数|
|person_change | 修改个人信息 |
| delete| 删除用户|
| fuzzy_search | 模糊搜索 参数key |
| recommend| 推荐 |
| adminlist| 返回管理员列表|
| permission|管理权限和权限组|
